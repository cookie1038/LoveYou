# LoveYou
python桌面壁纸和温馨提示

### python实现功能
1.连接数据库

2.爬取土味情话和天气预报（根据ip地址自动查询本地地址）

3.结合土味情话，设置桌面壁纸

4.结合土味情话、天气预报实现温馨提示

5.注册表实现自启动

6.Inno Setup打包工具打包
